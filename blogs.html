  • HTML: HyperText Markup Language

    September, 2022

    HTML or HyperText Markup Language is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance(CSS) or functionality(JavaScript).

    An HTML element is set off from other text in a document by tags, which consist of the element name surrounded by "<" and ">" . The name of an element inside a tag is case insensitive. That is, it can be written in uppercase, lowercase, or a mixture. For example, the <title> tag can be written as <Title>, <TITLE>, or in any other way. However, the convention and recommended practice is to write tags in lowercase.

    An HTML document consists of a certain structure to it:
    It begins with the <html> tags. Inside comes the <head> tags which contain the <title> tags and finally the <body> tag that contains all the content of the webpage. All these tags are also accompanied by their closing tags. Closing tags are the same as their counterparts other than the difference that they start with "/". eg. </html>, </head>, </body>.

    An HTML document is also where the CSS and Javascript files or functionalities are loaded\added.

  • Cascading Style Sheets (CSS)

    September, 2022

    Cascading Style Sheets are used to style and set the layout of web pages â€” for example, altering the font, color, size, and spacing of the content, splitting it into multiple columns, or adding animations and other decorative features. It is generally used by adding it to an HTML document to style it as you see fit. Each CSS property has a set of valid values, defined by formal grammar, as well as a semantic meaning, implemented by the browser. These values and properties specify the appearance of how, where, what color an element holds.etc, on a webpage.

    Another important facet of CSS is the box model. When laying out a document, the browser's renders each element as a rectangular box according to the standard CSS box model. CSS determines the size, position, and properties (color, background, border size, etc.) of these boxes. Every box is composed of four parts (or areas), defined by their respective edges: the content, padding, border, and margin :

    1. The content area, contains the content of the element, such as text, an image, or a video player. It often has a background color or background image.
    2. The padding area, extends the content area to include the element's padding.
    3. The border area, extends the padding area to include the element's borders.
    4. The margin area, extends the border area to include an empty area used to separate the element from its neighbors.

    The understanding and implementation of these concepts is extremely crucial to being a good designer.
